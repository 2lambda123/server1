Sub Catagory Container Oriented Functions
part of LuaScriptInterface

This sub catagory contains functions which returns data related to Containers. Examples: 

	-- Example 1
	function onUse(cid, item, frompos, item2, topos)
		if isItemFluidContainer(item) == 1 then
			-- script here
		end
		return 1
	end
	
	-- Example 2
	function onUse(cid, item, frompos, item2, topos)
		local container = doPlayerAddItem(cid,1987,1)
		doAddContainerItem(container,2006,7)
	end

 
Function Parameter Summary
uid
          +--Used when refering to the unique id of a creature. After obtaining a player/creature with a function such as getPlayerByName(name) or getThingfromPos(pos), you can use certain functions on that player/creature by setting the parameter of uid to "yourtempvariable".uid or just "function".uid
cid
          +--Used when refering to the player stated in the primary function. (The one doing that action) You can use certain functions with cid by setting the parameter of uid to cid.
itemid
          +--Used when refering to a specific item type. The value corispondes with the id of the item to be added/deleted.
slot
          +--Used when refering to a specific slot in your character.(backpack slot, sword slot, ect.) -- add the specific slot id's for ref. --
count
          +--Used when refering to a amount of a object, when stackable
subtype
          +--Used when refering to a item's subtype. It is used for example when changing a containers fluid type.
 

Function Detail 

isContainer(uid)
This function returns either 1 or 0 based on if the item located in uid is a container or not.
Usage : Used when checking if a item is a generic container. That includes fluid containers or item containers.
getContainerSize(uid)
This function returns the size of a container. The size is how many slots are in the container.
The returned value is a integer.
getContainerCap(uid)
This function returns the capacity of the requested container object.
The returned value is a integer.
getContainerItem(uid, slot)
This function returns the item in the requested slot of the item container object.
Usage : Commonly used when checking if a certain container slot has a certain item in it.
doAddContainerItem(uid, item, count or subtype)
This function adds a item into a container.
Usage : See example at top of page.
isItemContainer(itemid)
This function returns either a 1 or 0 based on if the requested itemid is a container that holds items.
Usage : Used as a container check.
isItemFluidContainer(itemid)
This function returns either a 1 or 0 based on if the requested itemid is a container that holds liquids.
Usage : Used as a container check.
